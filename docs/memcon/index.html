05
Install

One line. Five minutes.

Detects your RAM. Pulls the right local LLM. Registers itself with Claude Desktop.

The bootstrap script installs Docker if missing, brings up Qdrant + Ollama, picks a model from llama3.2:1b → qwen2.5-coder:32b based on machine size, then patches your claude_desktop_config.json so the MCP server is wired up on Claude's next restart.

  • Docker · Python 3.10+ required
  • ~8 GB disk · ~6 GB RAM for the default model
  • Re-runs are idempotent · upgrades in place
