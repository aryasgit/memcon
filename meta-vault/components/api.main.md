# api.main

**FastAPI app at :8000.**

Routes for `/ui` (the dashboard), `/memory/ask`, `/memory/query`,
`/memory/recent`, `/memory/save`, plus stats. Used by the
[[VS Code extension]] (which talks to it over HTTP from the editor process)
and the browser dashboard.
